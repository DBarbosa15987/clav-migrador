# Migrador CLAV

O migrador CLAV é o software que realiza a migração dos dados do CLAV vindos da Administração Pública. Este software é composto migrador de dados e por uma aplicação web que permite ao utilizador interagir com o backend.

## Dependências do Python

* pandas
* flask
* nanoid
* rdflib
* openpyxl

Para instalar as dependências:

```shell
pip install -r requirements.txt
```

## Como correr o programa

```shell
python run.py
```
